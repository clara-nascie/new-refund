# 🛠️ Stacks e Ferramentas Utilizadas

O projeto utiliza tecnologias modernas e de alta performance no ecossistema JavaScript para garantir agilidade no desenvolvimento e otimização em produção.

## 📦 Core & Frameworks
* **React 19:** Última versão do React, com melhorias de performance, novo compilador nativo e melhorias na hidratação.
* **Vite 8:** Ferramenta de build rápida e servidora de desenvolvimento que utiliza ESM nativo para HMR instantâneo.

## 🎨 Estilização
* **Tailwind CSS v4:** Framework utilitário de CSS reescrito para maior velocidade, com configuração baseada em arquivos CSS (@theme) em vez de arquivos JavaScript de configuração.
* **Clsx + Tailwind-Merge:** Usado na função utilitária `classMerge` para unir classes CSS condicionalmente e resolver conflitos de classes de maneira inteligente.

## 🚦 Roteamento
* **React Router v7:** Sistema de roteamento padrão para aplicações React, permitindo controle declarativo por estado e navegação fluida.
